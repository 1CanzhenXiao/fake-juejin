# 字节前端项目

#### 介绍
此项目为仿掘金网页版首页，是第四届字节跳动青训营的项目作业。主要复现掘金网页版首页外观，主要功能，如悬停效果，虚拟列表，无限下滑加载，打开文章时内容实现代码效果高亮，目录随文章滚动而变化。
