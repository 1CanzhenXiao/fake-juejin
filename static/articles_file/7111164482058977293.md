>本菜鸡在重庆，工作5年

今天距离我结婚一周年还有5天，仔细想想好想很久没有回顾过去了。

2022年刚开始有了向要跳槽的想法，之前的公司不能算大厂但在重庆也算是数一数二。

开始跳槽的的时候我其实挺犹豫的

其实说是有跳槽的想法在今年过年的时候就有了，因为每年公司3月会有涨薪的机会，所以想着看看那能不能涨（其实还是不想走）。


没办法，人都是有舒适圈的，这家公司已经做了3年了，同事已经变成朋友了，项目是自己从0 搭建的，习惯了每天到点下班一分钟不耽搁。

但是俗话怎么说，人往高处走，水往低处流。每个人都向要更高的工资，更多的休息时间，更多的尊重。

## 说说开始面试的经历吧

从3月开始面试，刚开始只是抱着试一试的心态所以没有考虑工作地点和通勤时间。

- 第一家，重庆德勤。熟悉重庆的朋友应该都懂，德勤在重庆还是有一定竞争力的。外企，薪资开得起，福利比较高，但是可能出差，项目不同可能加班，项目可能是外包。面试过程到是没什么波折，就是问了问vue ES6 和一些css的东西，我觉得总体来说是好过的，工资开的是13k*13薪。

- 第二家，渝中一家上市公司，这家公司是群里的一个大佬内推的，面试也是一轮视频技术面，不过在视频的过程中做了一些题，也是比较经典的题目（事件循环，闭包，promise，基础偏多的题），还有稳了一些正则的问题，还好问的不深，不然真的两眼一抹黑。然后就是谈薪资，当时技术负责人还挺满意的，加上我坚持要15k，最后给的15k*14薪。

面过了之后已经想去提离职了，然后负责人还是劝我说找个离家近的，毕竟上下班时间太长很降低幸福感的。我也问了问我的同事（hxd）们，他们也在劝我找个近的，或者等涨，不给涨再走就是了。

时间到了4月，公司没涨。好吧，继续面试。

- 投了长安找人推，简历没过。字节找人推，简历没过。吉利，简历没过。

- 面了春秋航空，问的挺深的还有一些实际经验上的东西，本来以为回答的不咋样呢，还是过了。13k*13薪。

- 面了一家上市的做办公的企业，稳定发挥，加上我坚持要15，最后给的15k*14薪。**最后去的公司**

### 投了腾讯
- 面了第一轮，问的项目上的实现，加上一些基础，说我不太符合他们部门推到了其他部门。
- 第二轮，问的也是项目，加上一些计算机网络（滑动窗口，TCP，UDP），
- 第三轮，项目做过的难点、亮点，+一些性能优化，小程序和其他的一些底层原理，未来规划，
- 第四轮，人事面，问了问基本情况，加了下微信，工资要等第五轮定级完了再谈。
- 第五轮，面试委员，还是项目+之前一些问题。

谈薪资，我说我已经有15k的offer了但是最后说只能给12k。我说能不能争取一下呢，回我说争取不了，好吧，公司都有规定。

在我入职新公司一周后，HR说业务部门还想争取我，所最多还能争取1-2k。没法，缘分没到，期待下次吧。

其他一些重庆的公司比如猪八戒，啄木鸟等我也有过了，一般给的都是13k。没什么值得说的。

## 总结一下

- 其实很多时候，找工作就像找对象，双方看对眼了，缘分到了，自然就水到渠成。
- 基础真的很重要，很多时候能帮助你和其他前端er区分出来。
- 多总结，输出一些技术文章，可能一开始写的磕磕绊绊，但是长久来看是很有帮助的。
- 还有很重要的一点，前端不要局限在使用框架，格局可以打开一点，计算机网络，数据结构这些程序员必备的技能也需要会的。越往后走，这些原理方面的东西越能够帮你快速学习。
- 学习是一辈子的事情，要享受学习带来的长期满足。
- 工作和生活要平衡，下了班就快回家，不要卷了。我现在最大的成就 就是娶了一个好老婆~

##  未来的一些设想

打算在这个公司再做上3年左右
- 学习一些数据可视化的东西
- 做一些工程化的实施
- 做一些音视频相关的研究
- 加深一些node相关的理解
- 继续输出一些技术文章
- 团结同事不要卷
- 要一个宝宝
- 装修一下房子


最后，祝大家都能生活愉快，做有意义的事~
