# 在外企的一年

去年 8 月份入职现在这家公司，和上家单位相比不用出差、不用打卡、不用加班、没有严格的绩效考核，当然也没有年终奖🤣，终于下班的时候不是看着路灯而是看着夕阳。

自从 3 月份上海疫情爆发到现在一直居家办公，下周开始回办公室。疫情期间工会还发了两次大礼包，第一次是在死活抢不到菜的那段时间；后来已经快结束的时候发了第二次大礼包，第二次是苹果橙子可乐薯片饼干零食哈哈哈...

除此之后还收到了工会许多大件福利，我选的基本都是跟厨房相关的，摩飞速蒸锅、摩飞多功能锅、防疫大礼包，还有很多小件的礼物堆满了我办公室的柜子。

在个人提升上面，公司也组织了内部和外部的很多培训，每一位在培训上提问的同事基本都会获得一些小礼物，而我也非常幸运作为讲师参与了部门以及整个公司的分享，还收获了一件定制的 T shirt。

年假也很足啊，我下半年入职，入职三个月之后还有 5 天年假，休假也没有那么麻烦还需要想一堆的理由，直接提交申请发送 OOO 邮件就好了。

我所在的团队按照工作职能来划分是属于 APAC，但是大部分同时都在印度，只有两个人在上海，所以免不了跟印度同事开开会啥的，好在业务交集不是特别多，也就没有特别多的会。
